pelican-themes

描述

pelican-themes 是一个用来管理Pelican主题的命令行工具。

用法

pelican-themes [-h] [-l] [-i theme path [theme path ...]]
[-r theme name [theme name ...]]
[-s theme path [theme path ...]] [-v] [–version]

可选参数

  • -h,--help 显示帮助
  • -l, --list 显示已安装的主题
  • -i theme_path , --install theme_path 安装一个或多个主题
  • -r ...
